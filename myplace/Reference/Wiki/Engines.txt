== # Perl Based == 
* [http://ikiwiki.info/ ikiwiki], a "wiki compiler" by Joey Hess - can use Subversion or git as the backend storage mechanism.
* [http://mojomojo.org/ MojoMojo] is an open-source web2.0 wiki built on top of the Catalyst web framework, blending the functions of wikis, CMSes and blogs
* [http://code.google.com/p/noosphere/source/browse/wiki/Noosphere.wiki?spec=svn2&r=2 Noösphere]], the engine for PlanetMath.
* [http://www.november-wiki.org/ November], a wiki engine written in Perl 6.
* [http://www.daemon.de/PodWiki PodWiki] is a wiki which supports multiple markup languages, primarily Perl POD.
* [http://www.socialtext.com/ Socialtext] is an enterprise wiki and weblog partially derived from open-source Kwiki. Socialtext is available as a hosted service, or a dedicated hardware appliance.
* [http://twiki.org/ TWiki] is a structured wiki, typically used to run a project development space, a document management system, a knowledge base, or any other groupware tool. Also available as a VMware appliance.
* [[.UseModWiki]] (Clifford Adams, 2000) is a clone of AtisWiki.
** [[.Oddmuse]] is a fork of [[.UseModWiki]].
* [http://c2.com/cgi/wiki WikiWikiWeb] (Ward Cunningham, 1994)
